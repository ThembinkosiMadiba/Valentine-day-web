## About This Project

This website takes you on an emotional journey:

- **Birthday Celebration** - A stunning intro with animated text and glowing effects
- **Love Letter** - An interactive envelope revealing a personal letter to MaMKhize
- **Crash Scene** - A creative "error" sequence that loops back to cherished memories
- **Custom Animations** - Smooth transitions and motion effects using Framer Motion

Built with love, featuring:

- React + TypeScript
- Framer Motion animations
- Tailwind CSS styling
- Custom fonts (Playfair Display, Great Vibes, Lato, Nunito)

## Running the Code Locally

**Install dependencies:**

```bash
npm i
```

**Start the development server:**

```bash
npm run dev
```

The site will open at `http://localhost:5173` (or the next available port).

## Project Structure

- `BirthdayIntro.tsx` - Animated birthday greeting scene
- `LetterScene.tsx` - Interactive envelope and love letter
- `CrashScene.tsx` - Creative transition effect
